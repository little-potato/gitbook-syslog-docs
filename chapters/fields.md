## Syslog 核心字段

### 字段说明
| 字段名       | 类型     | 描述               | 示例值                      |
|--------------|----------|--------------------|----------------------------|
| `priority`   | number   | 日志优先级级级级    | `34`                       |
| `timestamp`  | string   | 事件时间戳         | `2025-01-01T12:00:00Z`     |

### 原始日志示例
```syslog
<34>1 2025-01-01T12:00:00Z server1 sshd 12345 - Failed password for user 'admin'
```
